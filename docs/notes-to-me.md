# Virtuoso Notes

The right side will have 2 tabs
Composer / Conductor

Composer will be able to record a composition
There will be a list of existing compositions
The composer can click on a composition from list to view in the side menu too 
The composer can delete a composition
the composer can view a composition (as mentioned)
the composer can click on send stanza from a composition to load them into the main UI
the composer can remove actions and stanzas from a composition
the composer can create new compositions
the composer can record onto existing compositions
the composer can shuffle the order of actions via drag and drop in existing compositions

the conductor
Can export compositions as performance files
Can play compositions as performances
Can view performance results in the side menu too
